# Vision

copot is a modular website framework built to support:

- Company Profile
- Blog
- News Portal
- Knowledge Base
- Asset Management
- Workflow Automation
- Internal Dashboard
- Catalog
- Online Store

Core Philosophy:

Core System
+
Installable Modules
+
Installable Themes

The framework should be deployable on:

- Shared Hosting
- VPS
- Cloud Infrastructure

Long-term goal:

Create a flexible framework that can evolve from a simple website into a complete business platform.
